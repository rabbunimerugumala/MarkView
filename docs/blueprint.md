# **App Name**: MarkView

## Core Features:

- Live Markdown Editor: Real-time updates as the user types or pastes Markdown content in the textarea.
- Live Markdown Preview: Renders the Markdown content in real-time, reflecting the changes in the editor.
- Syntax Highlighting: Provides syntax highlighting for code blocks within the Markdown preview.
- Dark/Light Mode Toggle: Allows the user to switch between dark and light themes for optimal viewing experience.
- Copy to Clipboard: Button to copy the raw Markdown content from the editor to the clipboard.
- Clear Editor: Button to clear the Markdown textarea.

## Style Guidelines:

- Primary color: Slate blue (#64748B) for a developer-tool aesthetic.
- Background color: Light gray (#F9FAFB) in light mode, dark gray (#111827) in dark mode.
- Accent color: Teal (#2DD4BF) for links and interactive elements to provide subtle contrast.
- Font pairing: 'Inter' for body text and UI elements (sans-serif), and 'Literata' for rendered Markdown headings (serif).
- Code font: 'Source Code Pro' for code blocks in the markdown preview.
- Two-column layout: Editor on the left, preview on the right, responsive to screen size, stacking vertically on smaller screens.
- Minimal icons for theme toggle, copy, and clear buttons. To provide simple, functional controls that match the application's aesthetic.
